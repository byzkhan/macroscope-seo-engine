# Macroscope Brand Context

## What Macroscope Is
Macroscope is an AI-powered code review platform that integrates with GitHub and GitLab. It analyzes pull requests using large language models to catch bugs, enforce coding standards, and accelerate PR workflows.

## What Macroscope Is Not
- Not a linter or static analysis tool (though it complements them)
- Not a replacement for human code reviewers
- Not a code generation tool
- Not a CI/CD platform

## Target Audience
- Engineering leaders (VPs, Directors, Staff+) evaluating developer tooling
- Senior software engineers responsible for code quality on their teams
- DevOps/platform engineers building CI/CD pipelines and developer workflows
- Engineering managers tracking team productivity and code quality metrics

## Core Value Proposition
Macroscope is the intelligent layer on top of your PR workflow. It catches the bugs that linters miss — logic errors, security vulnerabilities, race conditions, and incomplete error handling — by understanding what your code is supposed to do, not just how it's structured.

## Key Differentiators
1. **AI-powered review comments** — contextual, specific, actionable feedback on every PR
2. **Team pattern learning** — adapts to your team's conventions and codebase over time
3. **GitHub/GitLab native** — integrates into existing workflows without new tools to learn
4. **Beyond pattern matching** — semantic code understanding, not rule-based checks
5. **Configurable review guidelines** — teams define what matters; Macroscope enforces it

## Tone and Voice
- **Authoritative but approachable** — we know this space deeply, but we don't talk down
- **Technical but clear** — precise language, concrete examples, no jargon for jargon's sake
- **Opinionated but evidence-based** — we take positions and back them with data
- **Confident, not arrogant** — state things directly, don't hedge unnecessarily
- **Practical, not theoretical** — focus on what teams can actually do

## Competitive Positioning
We compete with: Code Climate, SonarQube, Codacy, and emerging AI review tools (GitHub Copilot code review). Our differentiation is depth of AI analysis, team learning, and developer experience.

We do NOT compete with: linters (ESLint, Pylint, RuboCop), test frameworks, or CI/CD platforms. These are complementary tools.
